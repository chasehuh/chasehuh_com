# chaewon.me

Personal website for Chaewon.

- Live: https://chaewon.me (Vercel)
- Tech: Next.js (App Router), React, TypeScript, Tailwind

## Local development

Requirements:
- Node.js >= 20
- pnpm

Install + run:

```bash
pnpm install
pnpm dev
```

Build:

```bash
pnpm build
pnpm start
```

## Notes

- `pnpm dev` and `pnpm build` run `scripts/generate-blog-data.js` first.
